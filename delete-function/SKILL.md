# 删除源代码中指定的函数定义

- 删除工程中的函数[$ARGUMENTS]

## 执行步骤

1. **搜索函数定义**: 使用 Grep 工具在代码库中搜索函数名，查找函数定义位置
   - 搜索模式: `^func\s+函数名` (Go语言)
   - 搜索模式: `^function\s+函数名` (JavaScript)
   - 搜索模式: `^def\s+函数名` (Python)
   - 搜索模式: `^fn\s+函数名` (Rust)

2. **搜索函数引用**: 使用 Grep 工具搜索所有调用该函数的地方
   - 搜索模式: `函数名\(` (注意包含括号，避免匹配类型定义)
   - 排除函数定义行，只查找调用位置

3. **识别函数边界**: 根据语言语法确定函数的完整范围
   - Go: 从 `func` 开始到对应的 `}` 结束
   - JavaScript: 从 `function` 开始到对应的 `}` 结束
   - Python: 从 `def` 开始到下一个同级别 `def` 或类结束

4. **处理引用**: 分析每个引用点
   - 如果引用的函数也是仅被被删除函数调用，一并删除
   - 如果引用点会导致编译错误，询问用户如何处理

5. **执行删除**: 使用 Edit 工具删除函数定义及无效的引用函数

## 注意事项
- 如果函数是类的方法，需要同时考虑缩进
- 构造函数、析构函数需要特殊处理
- 导出的函数(Export)需要检查是否有外部调用
